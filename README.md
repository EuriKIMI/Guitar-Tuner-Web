# Studio Guitar Tuner

Studio Guitar Tuner is a browser-based practice app that combines live pitch detection with extra tools for timing and harmony work.

## Features

- Live microphone-based pitch detection
- Automatic and manual tuning modes
- Guitar tuning presets including Standard, Drop D, Half-Step Down, DADGAD, Open G, and Open D
- Reference tone playback
- Metronome with subdivision controls and tap tempo
- Chord guide with fretboard note mapping
- Progression practice playback
- Studio-style responsive interface

## Tech

- HTML
- CSS
- JavaScript
- Web Audio API

## How to use

1. Open `index.html` in a modern browser.
2. Allow microphone access when prompted.
3. Click `Start Listening` to begin pitch detection.
4. Use automatic mode for nearest-note detection or manual mode to lock onto a specific string.
5. Use the metronome, chord guide, and progression tools for extra practice support.

## Notes

- Microphone access is required for live tuning.
- For the best results, pluck one string clearly and reduce background noise.
